# triple-a
Article Analysis Assistant


# Use 

```
python -m venv venv
```

```
.\venv\Scripts\activate
```


```
pip install poetry
```

```
poetry install
```

```
poetry run python triplea/cli/main.py 
```

# testing

```
poetry run pytest
```
